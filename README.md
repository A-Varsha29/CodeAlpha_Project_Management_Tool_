# CodeAlpha_Project_Management_Tool

This is a web-based Project Management Tool designed to streamline team collaboration, task tracking, and project oversight. Developed using the Django framework, it provides a structured environment for managing project data and user authentication.

Features
Task Tracking: Efficiently manage project milestones and individual tasks.

User Authentication: Secure access control for team members.

Collaborative Workflow: Built to facilitate organized team communication and progress monitoring.

Tech Stack
Framework: Django (Python)

Database: SQLite3

Development Environment: Python virtual environment (venv)
